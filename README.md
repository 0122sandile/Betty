# Betty

### Run
Run the following command to check if your code/doc fits the Holberton Style (mostly inspired from the Linux Kernel style):
```
./checkpatch.pl --no-tree --terse -f <your_file(s)>
```
```
./kernel-doc.pl -list <file>
```

### Atom editor
